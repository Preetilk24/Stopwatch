# Stopwatch

## Features  
- **Start, Stop, and Reset** functionality  
- Real-time display of elapsed time  
- Clean and user-friendly interface  
- Responsive design for use on multiple devices


## Technologies Used  
- **HTML**: Provides the structure of the application.  
- **CSS**: Ensures styling and responsiveness for different devices.  
- **JavaScript**: Implements the stopwatch logic and interactivity.  

## How It Works  
- **Start**: Begin counting time by clicking the "Start" button.  
- **Stop**: Pause the stopwatch by clicking the "Stop" button.  
- **Reset**: Reset the time to `00:00:00` by clicking the "Reset" button.  

## Feature Enhancements  
- Add a **lap time feature** to track multiple intervals.  
- Save **timing history locally** for reviewing past sessions.
